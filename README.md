Please watch <a href="https://youtu.be/tYtp1F1NswY" target="_blank">this video</a> to get started.

Controls:
- `SPACE`: random action
- `S`: best action
- `R`: reset environment
- `Q`: quit

This game is built on top of <a href="https://www.gymlibrary.dev/" target="_blank">OpenAI Gym</a>.
